# 查找XML中的节点信息
```python
## 导入模块

## 获取root节点
```

# 获取某路径下所有的文件夹名称
```python
## 导入模块
import os
## 获取需要指定路径下所有的文件夹
def getDir(path):
		current_files=os.listdir(path)
		for file_name in current_files:
			full_file_name=os.path.join(path,file_name)
			if os.path.isdir(full_file_name):
				all_business.append(full_file_name)
		return all_business
```
